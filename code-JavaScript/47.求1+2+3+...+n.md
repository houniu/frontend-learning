## 题目描述

求 1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case 等关键字及条件判断语句（A?B:C）。

## 解题思路

这里用到了很简单的数学知识，等差数列的前 n 项和：`(q1+qn)*n/2`

公式很好记：类似于`梯形的面积=上底+下底*高/2`（想象成每一行有递增个元素，堆成了一个梯形，那么所有行的和也就是梯形的面积）

## Code

```javascript
function Sum_Solution(n) {
  // write code here
  var ans = ((1 + n) * n) / 2;
  return ans;
}
```
